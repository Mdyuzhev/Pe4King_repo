# Changelog

Все значимые изменения проекта документируются в этом файле.

## [1.0.0] - 2025-01-29

### VS Code Extension

#### Added
- Управление коллекциями API-запросов
- Test Runner с поддержкой Postman-сниппетов
- Экспорт в pytest, REST Assured, k6
- EVA модуль анализа качества тестов
- Интеграция с TestIT (Excel экспорт)
- Pre-request и Post-response скрипты

### IntelliJ IDEA Plugin

#### Added
- Просмотр и редактирование коллекций
- EVA анализ с UI панелью
- Генерация тест-кейсов
- Экспорт в различные форматы
